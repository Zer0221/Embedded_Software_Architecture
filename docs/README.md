# 软件文档目录

本目录包含了电梯视频对讲系统软件的核心文档。为了便于查阅，文档已经按照不同的类别进行了分类整理。

## 文档目录结构

```bash
docs/
  ├── architecture/  # 软件架构相关文档
  ├── guides/        # 用户指南和使用手册
  ├── api/           # API接口文档
  └── backup/        # 旧版文档备份（仅供参考）
```

1. 所有文档已经按照类别整理到对应的子目录中，请在相应的目录中查找所需文档。
